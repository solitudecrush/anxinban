## 补充说明

1. **血压字段差异**：Web 端使用字符串 `bp`（如 `"125/82"`），App 端 API 返回拆分为 `systolic` 和 `diastolic` 两个整型字段。建议后端同时支持两种格式，或统一为拆分的整型字段。
2. **老人 ID 绑定**：App 端当前固定绑定 `id = 100` 的老人（"曾姐"），Web 端管理全部老人数据。
3. **时间格式**：Web 端主要使用 `"yyyy-MM-dd HH:mm"` 字符串；App 端 API 主要使用 ISO8601 格式（`DateTime.now().toIso8601String()`）。
4. **监控权限**：`cameraAuthUntil` 为 0 表示无权限，非 0 值表示权限到期的时间戳。
5. **状态编码统一建议**：Web 端和 App 端的工单/申请状态编码存在差异，建议后端统一状态枚举值。
